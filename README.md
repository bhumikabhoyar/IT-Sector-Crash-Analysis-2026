# 📉 IT Sector Crash Analysis — The Accenture Connection

## 🎯 Project Overview
Event-driven stock market analysis of June 2026 Indian IT 
sector crash triggered by Accenture's Q3 guidance cut.

## 🔍 Key Findings
- ACN crashed **-17.97%** on Jun 18, 2026
- Indian IT avg fall: **-5.37%** next day
- Total MCap eroded: **₹4,40,864 Crore**
- Infosys hit **5-year low**, TCS hit **6-year low**
- ACN volume: **41M shares** (7x normal)

## 💡 Core Insight
Accenture acts as a **bellwether** for global IT demand.
Its guidance cut triggered immediate selloff in Indian IT — 
proving high correlation between global and Indian IT sectors.
